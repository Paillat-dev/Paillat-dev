Pythology

Pythology is a YouTube channel about Python. But it has a particularity: it is 99% automated. It uses a custom script to generate the videos, thumbnails, titles and descriptions thanks to multiple AI technologies and different APIs. The videos are then uploaded to YouTube automatically. The only needed manual work is to write make the video public on the YouTube Studio interface, because YouTube doesn't allow to do it with the API.

Smslex.ch

Smslex.ch is a Swiss website built for a client. Its goal is to summarize judgments of the Swiss Federal Court in an sms' length. It is built with wordpress with custom css and javascript, to allow the client to easily update the website.
